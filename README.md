# AI-ML Internship Tasks

## Overview
This repository contains various tasks completed during an AI and ML internship. Each notebook focuses on different aspects of machine learning and data analysis.

### Iris Intern (iris_intern.ipynb)
- **Description**: This notebook explores the Iris dataset, a classic dataset for classification tasks in machine learning.
- **Objectives**: To classify the Iris species based on sepal and petal measurements.
- **Methodologies**: Various classification models such as k-Nearest Neighbors (k-NN) and Support Vector Machines (SVM) were employed. Data preprocessing included normalization and exploratory data analysis.
- **Results**: The classification models provided insights into species differentiation, with accuracy scores showcased.

### StockPrice (StockPrice.ipynb)
- **Description**: This notebook involves predicting stock prices using historical data.
- **Objectives**: To forecast future stock prices based on historical trends.
- **Methodologies**: The analysis utilized time series forecasting methods including ARIMA and LSTM neural networks.
- **Results**: Key predictions and visualizations of stock trends were generated, demonstrating the effectiveness of the applied methodologies.

### Task 4 Health Bot (Task4HealthBot.ipynb)
- **Description**: This project implements a health chatbot that provides advice based on user queries.
- **Objectives**: To develop an interactive bot capable of answering health-related questions.
- **Methodologies**: Natural Language Processing (NLP) techniques and APIs were utilized to interpret user input and provide accurate responses.
- **Results**: The bot's performance was evaluated on user interaction metrics, showcasing its utility in providing health advice.

### Heart Disease (heartDisease.ipynb)
- **Description**: This notebook focuses on predicting heart disease based on patient data.
- **Objectives**: To identify risk factors and predict heart disease in patients.
- **Methodologies**: Regression and classification algorithms, including logistic regression and decision trees, were used along with feature importance analysis.
- **Results**: The analysis revealed significant predictors of heart disease, contributing to early risk identification.

## Conclusion
This repository captures various analyses and models developed during the internship, highlighting the practical application of machine learning techniques in diverse domains.
